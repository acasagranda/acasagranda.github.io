A Japanese-style puzzle that strengthens observational and problem solving skills.

Each puzzle has a unique solution which does not require any guess work to solve.


DIRECTIONS:
Fill all white cells in the grid with either an X or an O so that there are never 3 consecutive X's or O's in any row, column or diagonal.
